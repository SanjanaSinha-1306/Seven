# SEVEN AI 🧠✨

**Seven** is a sleek, persona-driven AI chat application designed with a "vibe-first" philosophy. Built with a modern tech stack and a neon-fusion aesthetic, it allows users to interact with various AI personalities—ranging from a tech-savvy "Nerd" to a "Sarcastic" genius—within a fully responsive, professional-grade interface.

## 🚀 Key Features

* **Persona-Driven Interactions**: Choose from 7 unique personas (Nerd, Genius, Love, Sassy, Gamer, Mystic, Hype Man) that adapt their slang and tone to your vibe.
* **Neon-Fusion UI**: A custom-designed dark mode interface using a unique "fused" gradient of Sky Blue and Purple.
* **Fully Responsive**: Seamless mobile experience with a dedicated slide-in hamburger menu and optimized touch targets.
* **Persistent Memory**: Local storage integration ensures your chat history and active sessions stay saved even after a refresh.
* **Smart UX**: Includes a vertical splash screen, "Empty State" background prompts, and an intelligent scroll viewport that prevents message clipping.

## 🛠️ Tech Stack

* **Frontend**: HTML5, Tailwind CSS, Lucide Icons.
* **Logic**: Vanilla JavaScript (ES6+).
* **API**: Groq Cloud SDK (Llama-3.3-70b-versatile).
* **Styling**: Custom CSS for neon-glow effects and "fused-text" gradients.

## ⚙️ Setup & Installation

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/your-username/seven-ai.git](https://github.com/your-username/seven-ai.git)
    ```
2.  **Add your API Key:**
    Open `app.js` and replace the `apiKey` value in the `state` object with your [Groq API Key](https://console.groq.com/keys).
3.  **Add Assets:**
    Place your `logo.png` and `favicon.ico` in the root directory to fix any 404 errors.
4.  **Launch:**
    Open `index.html` using a local server (like VS Code Live Server) to start vibing.
